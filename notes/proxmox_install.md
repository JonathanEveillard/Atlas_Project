# Installing Proxmox [Bare Metal]

## Current Specs Devices used for Proxmox
- Intel I7 (Central Processing Unit)
- 1TB (Storage)
- 128GB (RAM)
- RTX 3060 (Graphical Unit)

# ISO Installation

[ISO](https://proxmox.com/en/downloads/proxmox-virtual-environment/iso)

[RUFUS](https://rufus.ie/en/)
[BALENAETCHER](https://etcher.balena.io/)

To get started with with the Proxmox Installation, we will have to install the `ISO` Image first on a USB stick with atleast a minimum of 2-4gb of storage. Make sure you are using Rufus or BalenaEtcher.

Similar to any operating system, you will have the installation screen which includes, networking configuration, user, storage configuration. Usually it is automatically set for you as default but if it's not take some time to go through it. It's not long nor difficult to understand, hence this note is relatively short.
